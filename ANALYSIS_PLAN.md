# Analysis Plan

## Objective
Understand user behavior and identify opportunities to improve conversion and retention.

---

## Key Metrics to Analyze

### 1. Funnel Metrics
- Total Users
- Product Views
- Add to Cart Events
- Purchases
- Conversion Rates:
  - View → Cart
  - Cart → Purchase

---

### 2. User Behavior
- Number of events per user
- Average sessions per user
- Event distribution

---

### 3. Retention Metrics
- Cohort retention (monthly)
- Returning user %

---

## Hypotheses

1. Majority drop-off happens between product view and add-to-cart
2. Only a small percentage of users return after first interaction
3. Users who add to cart are highly likely to purchase

---

## Analysis Approach

1. Build funnel using SQL
2. Calculate user-level metrics
3. Perform cohort retention analysis
4. Validate hypotheses using data
